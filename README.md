# 📋 Kanban Task Management Board

A simple, responsive Kanban-style task management board built with plain HTML, CSS, and JavaScript. Tasks are saved in the browser using `localStorage`, so your board persists even after refreshing the page.

🔗 **Live Demo:** [bhadaneanuja5-bit.github.io/task-management-board](https://bhadaneanuja5-bit.github.io/task-management-board/)

# ✨ Features

- **Drag & Drop** — Move tasks between columns (To Do, In Progress, Done) with smooth drag feedback
- **Priority Tags** — Mark tasks as High, Medium, or Low priority
- **Due Dates** — Add and track deadlines for each task
- **Toast Notifications** — Get instant feedback when you add, edit, or delete a task
- **Responsive Design** — Works smoothly across different screen sizes and zoom levels
- **Local Storage** — All tasks are saved in your browser, no backend required

# 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- Browser `localStorage` API

# 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/bhadaneanuja5-bit/task-management-board.git
   ```
2. Open `index.html` in your browser — that's it, no build steps or dependencies needed!
